# Invest Pro 2.0 🚀

**Invest Pro 2.0** — це комп'ютерна система підтримки прийняття рішень (Decision Support System - DSS) у сфері інвестицій. Проект розроблений для автоматизації процесу аналізу акцій, використовуючи математичне моделювання та алгоритми штучного інтелекту для обробки природної мови.

Система дозволяє користувачеві формувати інвестиційний портфель на основі індивідуальних переваг (ризик, прибутковість, вартість, дивіденди), аналізуючи реальні біржові дані та новинний фон у режимі реального часу.

## 🌟 Ключові наукові та інженерні рішення

1. **Математичне моделювання (AHP алгоритм):**
   Використовується метод аналізу ієрархій (Analytic Hierarchy Process) Томаса Сааті. Система динамічно генерує матрицю попарних порівнянь на основі налаштувань повзунків користувача, обчислює нормалізовані власні вектори (ваги критеріїв) та вирішує багатокритеріальну задачу вибору оптимальних активів.
   
2. **Обробка природної мови (NLP Sentiment Analysis):**
   Інтегровано лексико-евристичну модель ШІ **VADER** (Valence Aware Dictionary and sEntiment Reasoner). Алгоритм аналізує свіжі новинні заголовки про компанії та трансформує лінгвістичний контекст у кількісні показники (compound score) для оцінки настроїв ринку (Позитив, Негатив, Нейтрально).

3. **Нормалізація гетерогенних даних та ранжування:**
   Обробка фінансових метрик, поділених на стимулятори (маржа прибутку, дивіденди) та дестимулятори (Beta-ризик, P/E). Алгоритм використовує математичні перетворення для нормалізації даних та векторне множення для фінального ранжування.

4. **Алгоритмічна оптимізація:**
   Для роботи з часовими рядами (time-series data) у реальному часі та обходу обмежень зовнішніх API впроваджено стратегії **In-Memory кешування** (Redis / Django Cache), що значно знижує обчислювальне навантаження.

## 🛠️ Технологічний стек

- **Backend:** Python, Django 6.0
- **Data Science & Math:** NumPy, Pandas
- **Фінансові дані:** `yfinance` (Yahoo Finance API)
- **AI / NLP:** `vaderSentiment`
- **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript
- **Візуалізація:** ApexCharts (радарні та кругові діаграми)
- **База даних:** SQLite3 (за замовчуванням), підтримка PostgreSQL
- **Сервер / Деплой:** Gunicorn, WhiteNoise, ASGI/WSGI

## 🚀 Основний функціонал

* **Динамічний пошук тікерів:** Живий пошук акцій з Yahoo Finance.
* **Каталог акцій:** Попередньо налаштовані категорії ("Техніка та IT", "Енергія", "Фінанси" тощо).
* **Гнучке налаштування стратегії:** Інтерактивні повзунки для налаштування важливості критеріїв за методом AHP.
* **Розподіл бюджету:** Автоматичний розрахунок кількості акцій для купівлі на основі виділеного бюджету з конвертацією валют (USD, UAH, EUR).
* **Аналітика та графіки:** Побудова SVG міні-графіків (sparklines), радарної діаграми лідерів та кругової діаграми розподілу бюджету.
* **Експорт:** Можливість вивантаження результатів у CSV або друку у PDF форматі.
* **Темна/Світла тема:** Сучасний UI з підтримкою збереження вибраної теми.

## ⚙️ Встановлення та запуск

1. **Клонування репозиторію:**
   ```bash
   git clone <URL_репозиторію>
   cd invest
   ```

2. **Створення та активація віртуального середовища:**
   ```bash
   python -m venv venv
   # Для Windows:
   venv\Scripts\activate
   # Для Linux/Mac:
   source venv/bin/activate
   ```

3. **Встановлення залежностей:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Застосування міграцій бази даних:**
   ```bash
   python manage.py migrate
   ```

5. **Запуск локального сервера розробки:**
   ```bash
   python manage.py runserver
   ```

6. **Доступ до додатку:**
   Відкрийте браузер та перейдіть за адресою: `http://127.0.0.1:8000/`

## 📁 Структура проекту

* `analyzer/` - Головний Django-додаток.
  * `views.py` - Контролери для обробки запитів та рендерингу UI.
  * `utils.py` - Ядро системи (клас `InvestmentAHP`), що містить усю бізнес-логіку: математику AHP, парсинг yfinance, аналіз NLP.
  * `urls.py` - Маршрутизація додатку.
  * `templates/analyzer/dashboard.html` - Основний інтерфейс користувача.
* `invest_project/` - Налаштування Django проекту.
* `computer_science_justification.md` - Наукове обґрунтування алгоритмічної складової (для захисту).
